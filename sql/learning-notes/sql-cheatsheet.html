SELECT Choisir les colonnes

Définit quelles colonnes tu veux voir dans les résultats.

SELECT *                      -- toutes les colonnes SELECT Name                  -- une colonne SELECT Name, City, Email  -- plusieurs
obligatoiretoujours en premier
DISTINCT Éliminer les doublons

Se place juste après SELECT. Affiche chaque valeur une seule fois.

-- Sans DISTINCT (doublons) SELECT City FROM Customers → Montreal, Montreal, Toronto -- Avec DISTINCT (unique) SELECT DISTINCT City FROM Customers → Montreal, Toronto
après SELECTélimine doublons
WHERE Filtrer les lignes

Filtre les lignes selon une condition. Contrôle QUELLES lignes apparaissent.

SELECT * FROM Products WHERE Price = 18         -- égal WHERE Price <> 18         -- différent WHERE Price > 10         -- supérieur WHERE City = 'Montreal' -- texte
optionnelfiltre les lignes
BETWEEN Dans un intervalle

Filtre les valeurs entre deux bornes. Les deux bornes sont incluses.

SELECT * FROM Products WHERE Price BETWEEN 10 AND 20 -- Équivalent à : WHERE Price >= 10 AND Price <= 20 -- Fonctionne aussi avec des dates WHERE Date BETWEEN '2024-01-01' AND '2024-12-31'
dans WHEREbornes inclusesAND obligatoire
IN Liste de valeurs

Raccourci pour plusieurs conditions OR. Spécifie une liste de valeurs acceptées.

SELECT * FROM Customers WHERE City IN ('Montreal', 'Toronto', 'Quebec') -- Équivalent à : WHERE City = 'Montreal' OR City = 'Toronto' OR City = 'Quebec'
dans WHEREremplace plusieurs OR
COUNT Compter les lignes

Fonction d'agrégation. Compte le nombre de lignes ou de valeurs non-nulles.

SELECT COUNT(*) FROM Customers → compte toutes les lignes SELECT COUNT(Email) FROM Customers → ignore les lignes sans email (NULL) SELECT COUNT(DISTINCT City) FROM Customers → nombre de villes uniques
après SELECTun argumentignore NULL
ORDER BY Trier les résultats

Trie les résultats selon une ou plusieurs colonnes, en ordre croissant ou décroissant.

SELECT * FROM Customers ORDER BY Name ASC -- A → Z (défaut) SELECT * FROM Products ORDER BY Price DESC -- plus cher en premier
optionnelASC ou DESCaprès WHERE
LIMIT Limiter les résultats

Limite le nombre de lignes retournées. Toujours en dernière position.

SELECT * FROM Products ORDER BY Price DESC LIMIT 5; -- seulement les 5 premiers
optionneltoujours en dernier
